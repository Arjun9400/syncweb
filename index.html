Connect to GitHub

This app syncs data to a JSON file in your GitHub repo — free forever. No backend needed.

Setup steps:
1. Create a GitHub repo (can be private)
2. Go to github.com/settings/tokens → Generate new token (classic)
3. Tick repo scope → copy the token
4. Enable GitHub Pages on your repo (Settings → Pages → main branch → /root)
